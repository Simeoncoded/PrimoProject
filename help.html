Help & FAQ

An NCR (Non-Conformance Report) documents deviations from standard procedures and specifications, detailing the issue to support corrective action.

Navigate to the "Create NCR Form" page, fill out the required fields, attach any supporting files, and submit the form to log a new NCR.

“CF” stands for “continuous feature.” This requirement is indicated when features need to be geometrically treated as a single feature.

All submitted NCRs can be found in the "NCR Log" section, where you can view and edit each NCR.

Notifications are shown in the "Notifications" section of the navigation bar. A badge displays the number of new updates.

To navigate the form using the keyboard, you can use the Tab key to move through the form fields. To go backward, press Shift + Tab. Use the Enter key to select dropdown options and submit buttons.

Fields marked with an asterisk (*) are required. You can also use screen reader software to announce these required fields, or check for any red error messages after attempting to submit the form without completing them.

On the Date field, you can manually type the date in the format MM/DD/YYYY or use a date picker to select the date. If you are using a screen reader, the date picker should be announced, allowing you to choose the date by navigating through calendar options.

No, the form will not submit if required fields are missing or if invalid information is entered. Error messages will appear next to the fields that need to be corrected. If you're using a screen reader, these messages should be read out to guide you in correcting the form.

You can access the help section by clicking the "Help" button or using the accordion feature to view additional information. Each question and answer provides detailed instructions on how to fill out the form correctly. If you still need assistance, contact support for further guidance.

You can add new suppliers on the create form by simply clicking the "Add Supplier" button located next to the supplier dropdown. This will allow you to input details for the new supplier and include them in your form submission.
